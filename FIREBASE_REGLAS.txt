REGLAS PARA FIRESTORE (pegar en Firebase > Firestore Database > Reglas)

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {

    function esAdmin() {
      return request.auth != null
        && request.auth.token.email == "austinscor12@gmail.com";
    }

    match /publicaciones/{id} {
      allow read: if true;
      allow create, update, delete: if esAdmin();
    }
  }
}

IMPORTANTE:
1. Publicar estas reglas.
2. En Authentication > Configuración > Dominios autorizados, agregar 127.0.0.1 si probás con Live Server.
3. Para entrar al panel: abrir la web y tocar "Panel privado" al final, o entrar a anidem_profesional.html#admin.
4. Primero iniciar sesión con Google usando austinscor12@gmail.com.
5. Luego tocar "Crear contenido inicial" una sola vez para cargar las publicaciones en Firestore.
6. Cuando tengas el Gmail de Cielo, cambiamos este correo en el HTML y en estas reglas.
